\section{Časový plán}
Plánujem pracovať iteratívne a to konkrétne tak, že celý projekt si rozdelím na menšie časti a každú z nich budem implementovať v určitom týždni semestra.
Zároveň si jeden deň vyhradím na code review práce z predchádzajúceho týždňa, aby som sa uistil, že všetko funguje správne a že kód je čistý a dobre zdokumentovaný.

\subsection*{5. týždeň semestra}
Implementácia základnej štruktúry stránky, navigácie, základných komponentov a lokalizácie (slovenčina, angličtina). 
Zároveň implementácia autentifikácie používateľa, teda registrácie, prihlasovania, zmeny hesla, zmeny údajov účtu a zrušenia účtu.

\hspace{-1.5em}\textbf{Odhad času:} približne 16 hodín
\begin{itemize}[itemsep=2pt]
    \item základný layout, navigácia, routing: 4 h
    \item základné reusable komponenty: 3 h
    \item lokalizácia: 2 h
    \item autentifikácia a formuláre: 5 h
    \item testovanie, code review a opravy: 2 h 
\end{itemize}

\subsection*{6. týždeň semestra}
Implementácia stránky pre zobrazenie nehnuteľností, recenzií a ich variantov vrátane dostupnosti a vytvorenia rezervácie.

\hspace{-1.5em}\textbf{Odhad času:} približne 16 hodín
\begin{itemize}[itemsep=2pt]
    \item zoznam nehnuteľností a variantov: 4 h
    \item detail variantu / nehnuteľnosti: 3 h
    \item zobrazenie recenzií a priemerného hodnotenia: 2 h
    \item vytvorenie rezervácie: 5 h 
    \item testovanie, code review a opravy: 2 h 
\end{itemize}

\subsection*{7. týždeň semestra}
Implementácia stránky pre správu rezervácií, histórie rezervácií a možnosti ich zrušenia. 
Zároveň implementácia možnosti vytvorenia recenzie po absolvovanom pobyte.

\hspace{-1.5em}\textbf{Odhad času:} približne 18 hodín
\begin{itemize}[itemsep=2pt]
    \item aktuálne rezervácie, ich história a zrušenie: 8 h
    \item zoznam recenzií s filtrovaním, pridanie recenzie, správa recenzií: 8 h
    \item testovanie, code review a opravy: 2 h 
\end{itemize}

\subsection*{8. týždeň semestra}
Implementácia stránky pre správu nehnuteľností, teda pridanie novej nehnuteľnosti, úpravu existujúcej a zoznam vlastnených nehnuteľností.

\hspace{-1.5em}\textbf{Odhad času:} približne 14 hodín
\begin{itemize}[itemsep=2pt]
    \item zoznam vlastnených nehnuteľností: 3 h
    \item formulár na vytvorenie nehnuteľnosti: 4 h
    \item editácia nehnuteľnosti a variantov: 5 h
    \item testovanie, code review a opravy: 2 h 
\end{itemize}

\subsection*{9. týždeň semestra}
Implementácia stránky pre správu používateľov vrátane zobrazenia používateľov, ich rezervácií, vlastnených nehnuteľností, súborov a možnosti úpravy ich údajov.

\hspace{-1.5em}\textbf{Odhad času:} približne 14 hodín
\begin{itemize}[itemsep=2pt]
    \item zoznam a detail používateľov: 6 h
    \item rezervácie / nehnuteľnosti / súbory používateľa: 6 h
    \item testovanie, code review a opravy: 2 h 
\end{itemize}

\subsection*{10 - 13. týždeň semestra}
Finalizácia projektu, optimalizácia výkonu, dokončenie testovania a príprava finálnej verzie.

\hspace{-1.5em}\textbf{Odhad času:} približne 28 hodín
\begin{itemize}[itemsep=2pt]
    \item integračné testovanie: 10 h
    \item opravy chýb, code review a refaktoring: 16 h
    \item príprava finálnej verzie: 2 h
\end{itemize}
